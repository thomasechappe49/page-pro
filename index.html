❓ Comment fonctionne cette page

Cette page est votre tableau de bord de démarrage : un seul fichier qui regroupe vos liens, rangés par catégories. Tout fonctionne hors ligne et vos réglages sont enregistrés dans ce navigateur.

Au quotidien

  • Ouvrir un lien : cliquez sur une tuile.
  • Filtrer : tapez dans le champ « Filtrer vos liens » (ou appuyez sur /) pour ne garder que les liens correspondants.
  • Recherche Web : choisissez un moteur puis cliquez sur « Recherche Web ».
  • Replier / déplier une catégorie : cliquez sur son titre. Une catégorie repliée se réduit à une étiquette en bas à droite ; recliquez dessus pour la rouvrir à sa place. Alt + clic sur un titre n'affiche que cette catégorie.
  • Thème : le bouton 🌙 / ☀️ bascule entre clair et sombre.

Mode Modifier (bouton ✏️)

Activez-le pour personnaliser la page ; recliquez (✓ Terminer) pour revenir à l'usage normal.

  • Catégories : « ＋ Nouvelle catégorie » en bas ; sur chaque en-tête, ✎ renomme, 🗑 supprime, et la pastille multicolore change la couleur de fond (⟲ remet par défaut).
  • Liens : « ＋ Lien » ajoute, ✎ renomme, ✕ supprime.
  • Déplacer un lien : glissez-le vers une autre catégorie ou pour le réordonner.
  • Ajouter depuis l'extérieur : glissez une adresse (barre d'adresse ou lien d'une autre page) directement sur une catégorie — possible même hors mode Modifier.
  • Apparence : « 🎨 Fond » (couleur générale), « 🖼️ Image » (image de fond, fichier ou URL), et « Cartes transparentes » pour laisser voir le fond.

Sauvegarde et transfert

  • Vos liens et réglages sont mémorisés localement : ils sont propres à cet ordinateur et à ce navigateur.
  • Pour une copie de sécurité ou pour transférer vers un autre poste, utilisez Exporter (CSV), puis Importer sur l'autre poste.
  • Vider les données du navigateur efface aussi vos personnalisations — pensez à exporter de temps en temps.
  • Exporter HTML génère un fichier autonome qui contient TOUT (liens, couleurs, image de fond, thème, état replié…) et se reconstitue à l'identique sur un autre poste.
  • ⬆️ GitHub publie cette page sur votre dépôt GitHub sous le nom index.html (utile avec GitHub Pages). À la première utilisation, indiquez le compte, le dépôt, la branche et un jeton d'accès personnel GitHub disposant du droit « Contents ». Le jeton reste uniquement dans ce navigateur et n'est jamais inclus dans le fichier publié. Alt + clic sur le bouton pour reconfigurer.

Astuce : refermez cette fenêtre avec ✕, la touche Échap, ou en cliquant à l'extérieur.
